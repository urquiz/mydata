Sobre este projeto

Este é um pequeno projeto que resolvi iniciar a fim de praticar uma das coisas que mais gosto, desenvolver. É realmente incrível, porque posso desenvolver algo do zero, enviar para as pessoas e elas podem interagir com tudo isso. No final é sempre gratificante quando finalizamos algo que iniciamos.

Porque Web Designer

A ideia é de criar algo que não existia anteriormente. A alegria de construir algo e vê-lo ganhar vida é o que torna o desenvolvimento da Web realmente incrível ao meu ver. Você começa a brincar com coisas brilhantes, novas e legais o tempo todo. Todos os dias se aprende algo novo que fascina ainda mais o trabalho. Isso para mim é fantástico.
